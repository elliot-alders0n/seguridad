
# Apache HTTP server benchmarking tool


## Instalación
```
apt-get install apache2-utils
```


## Ejecución de las peticiones en masa
```
ab –n <numero_peticiones> –c  <peticiones_concurrentes> <IPservidorWeb/URLservidorWeb>
```
